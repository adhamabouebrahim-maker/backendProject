1_Project Description:
This project is a student portal web application designed to manage and display student information such as names, IDs, departments, grades, and other academic details.
The goal of the project is to provide an easy and organized interface for users to view and manage student data.

2-Setup instructions:To set up the project, clone the repository and navigate to the backend folder. Install dependencies using npm install and start the server with node server1.js. The frontend can then connect to the backend APIs to display and manage data. For deployment, the backend can be hosted on Render or Railway, while the frontend can be hosted on GitHub Pages or any static hosting service.
